\# Hunter Tame Tracker (HTT)



Hunter Tame Tracker is a lightweight World of Warcraft addon that tracks all Hunter pet-taming unlocks (tomes, spells, and quest-based unlocks) in one clean, searchable UI.



FEATURES



\- Tracks 11 hunter taming unlocks

\- All / Missing / Unlocked tabs

\- Search (supports expansion aliases like MoP and BfA)

\- Progress tracking with completion bar

\- Copy Wowhead button (safe copy box, no taint issues)

\- Copy Waypoint button (only appears if TomTom is installed)

\- TomTom-compatible /way formatting



SLASH COMMANDS



/htt — Toggle the UI  

/htt version — Print addon version  



WAYPOINTS



If TomTom is installed, the Copy Waypoint button appears for entries that have coordinates.



Waypoints are formatted as:



/way X Y Name



Zone names are not included in the command (TomTom-safe).

INSTALLATION



1\. Download the latest release.

2\. Extract into:



World of Warcraft/\_retail\_/Interface/AddOns/



3\. Folder structure should be:



AddOns/HunterTameTracker/HunterTameTracker.toc  

AddOns/HunterTameTracker/HunterTameTracker.lua
